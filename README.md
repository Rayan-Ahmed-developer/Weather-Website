# 🌤️ Weather Website

A sleek, modern, and fully responsive Weather Web Application built using HTML, CSS (Glassmorphism), and JavaScript. This application allows users to check real-time weather conditions for various cities with a beautiful pixel-perfect user interface.

## ✨ Features
- 🔍 **Search City:** Quick weather search by entering the city name.
- 💧 **Humidity Tracker:** Displays precise humidity levels in percentage.
- 💨 **Wind Speed Indicator:** Shows live wind speed in Km/h.
- 📱 **Fully Responsive:** Optimised for all screen sizes, from small mobile devices to large desktops.
- 🎨 **Glassmorphism UI:** Modern UI design with blurred backdrops and vibrant gradients.

---

## 🛠️ Tech Stack Used
- **HTML5:** For semantic web structure.
- **CSS3:** Custom properties, Flexbox, Media Queries, and Glassmorphic styling.
- **JavaScript (ES6):** Dynamic UI updates and state management.
- **FontAwesome:** For high-quality vector icons.
